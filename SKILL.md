# 写作引用助手

当你写文章需要引用书籍原文来支撑观点时使用这个 skill。

## 输入

你只需要告诉我：

1. 你的文章（或粘贴文字）
2. 想引用哪本书/哪个作者
3. 想表达什么观点

## 工作流程

### 第一步：搜索原文

用 WebSearch 在网上搜索这本书/作者相关的内容，找到原文段落。

### 第二步：筛选原文

按照以下标准筛选：

- **传达核心观点** — 让读者快速理解你想表达什么
- **动词多、形容词少** — 动作导向，不要太煽情
- **深刻但易懂** — 外行也能看懂，不堆砌专有名词
- **20-50字** — 灵活控制，太短说不清，太长没重点

### 第三步：呈现选择

给我 2-3 段原文，每段包含：

- 原文内容
- 推荐程度（高/中/低）
- 推荐理由（为什么适合表达这个观点）

### 第四步：帮你做决定

如果找不到原文：

- 告诉我"找不到"
- 推荐其他"大神"级别的书（思想家、科学家、知名学者）
- 不推荐畅销书或"老头乐"类型的书
- 同样按照上面的筛选标准

## 输出格式

```
原文选项 1 【推荐程度：高】
--------------------------------
"原文内容..."

推荐理由：...

原文选项 2 【推荐程度：中】
--------------------------------
"原文内容..."

推荐理由：...
```

等你选择后，帮你格式化成引用格式放进文章里。
